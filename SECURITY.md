# Security Policy
## Supported Versions
Currently, only the latest deployment from the `main` branch is supported with security updates.
| Version | Supported |
| ------- | ------------------ |
| `main` (latest) | :white_check_mark: |
| older versions | :x: |
## Reporting a Vulnerability
We take the security of this project seriously. If you discover a security vulnerability within the Wheel of Names Mini App, please follow these steps:
1. **Do not open a public issue.** This ensures the vulnerability cannot be exploited by others before a patch is released.
2. Please report the vulnerability by navigating to the **Security** tab of this repository and clicking **Report a vulnerability** (if active), or contact the maintainer directly.
3. Provide a detailed description of the issue, including the steps to reproduce it and its potential impact.
All legitimate vulnerability reports will be reviewed and addressed as quickly as possible.
